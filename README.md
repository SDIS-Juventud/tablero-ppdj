# Tablero de seguimiento a Productos y Resultados — PPDJ 2019-2030

Seguimiento al Plan de Acción Indicativo de la Política Pública Distrital de
Juventud: metas programadas frente a lo ejecutado, por las 7 dimensiones de la
política, con vistas de Productos y de Resultados y análisis cualitativo por
indicador.

Tablero estático (HTML + Chart.js) con datos procesados en Python.

## Cómo verlo

Abrir `index.html` en el navegador, o visitar la versión publicada en GitHub
Pages.
